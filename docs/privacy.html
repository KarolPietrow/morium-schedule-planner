Polityka Prywatności aplikacji Morium

Ostatnia aktualizacja: 13.03.2026 r.

POSTANOWIENIA OGÓLNE

Niniejsza Polityka Prywatności określa zasady działania i przetwarzania informacji w aplikacji mobilnej "Morium" (zwanej dalej "Aplikacją"). Aplikacja jest niezależnym, nieoficjalnym projektem i nie jest w żaden sposób powiązana z systemem Moria, ani z Uniwersytetem Marii Curie-Skłodowskiej w Lublinie (UMCS).

GROMADZENIE I PRZECHOWYWANIE DANYCH

Aplikacja nie zbiera, nie przetwarza, nie przechowuje ani nie udostępnia żadnych danych osobowych na zewnętrznych serwerach. Wszystkie ustawienia i preferencje użytkownika (takie jak wybrane kierunki, przedmioty czy grupy) są zapisywane wyłącznie lokalnie, bezpośrednio w pamięci urządzenia użytkownika.

USŁUGI ZEWNĘTRZNE

Główną funkcją Aplikacji jest wyświetlanie planu zajęć. Aby to zrealizować, Aplikacja łączy się z publicznym, ogólnodostępnym API systemu Moria (https://moria.umcs.lublin.pl/api), pobierając stamtąd aktualne dane listy kierunków, oraz przedmiotów na kierunku wybranym przez użytkownika. Podczas nawiązywania tego połączenia, standardowe zapytania sieciowe, zawierające m.in. adres IP użytkownika, trafiają do serwerów UMCS. Deweloper Aplikacji nie pośredniczy w tym procesie i nie ma dostępu do tych danych.

WYMAGANE UPRAWNIENIA

Do poprawnego działania aplikacja wymaga jedynie dostępu do połączenia internetowego.

USUNIĘCIE DANYCH

Wszystkie dane przechowywane są lokalnie na urządzeniu użytkownika. Aby usunąć dane aplikacji, wystarczy odinstalować aplikację.

KONTAKT

W przypadku jakichkolwiek pytań, wątpliwości lub zgłoszeń dotyczących działania Aplikacji, kontakt jest możliwy pod adresem e-mail: karol.pietrow@icloud.com.

Morium App Privacy Policy

Last updated: 2026-03-13

GENERAL

This Privacy Policy sets out the rules for the operation and processing of information in the "Morium" mobile application ("App"). The App is an independent, unofficial project and is in no way affiliated with the Moria system or the Maria Curie-Skłodowska University in Lublin (UMCS).

DATA COLLECTION AND STORAGE

The App does not collect, process, store, or share any personal data on external servers. All user settings and preferences (such as selected fields of study, subjects, or groups) are saved only locally on the user's device.

THIRD-PARTY SERVICES

The primary function of the App is to display class schedules. To achieve this, the App connects to the public, open API of the Moria system (https://moria.umcs.lublin.pl/api) to fetch the current list of fields of study and the subjects for the specific program selected by the user. During this connection process, standard network requests, which include the user's IP address, are routed to the UMCS servers. The App's developer does not act as an intermediary in this process and has no access to this data.

REQUIRED PERMISSIONS

To function properly, the App requires access to an internet connection.

DATA DELETION

All data is stored locally on the user's device. To delete the App's data, the user has to uninstall the App.

CONTACT

If you have any questions, concerns, or reports regarding the operation of the App, you can contact the developer at karol.pietrow@icloud.com.
